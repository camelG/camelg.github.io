      # GraphQL の API 種類

      • Query (取得)
      • Mutation (更新)
      • Subscription (変更検知)

      # Schema 定義

      schema {
  query: Query
  mutation: Mutation
  subscription: Subscription
}
      • Shop オブジェクト定義
      • : は型定義
      • ! は NOT NULL

      ※ID 型がある

      スキーマ＆モデルセットして使う

      schema{
    type Shop {
        id: ID!
        name: String!
    }

    type Query {
        getShops: [Shop]!
    }
}

      クライアントから直接取り出す

      query {
  getShops {
    id
    name
  }
  getShop(id: "1") {
    id
    name
  }
}
